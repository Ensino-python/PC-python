Variável Booleana

Boolean é um tipo de variável diferente das vistas anteriormente, nela os valores possíveis são apenas VERDADEIRO ou FALSO. Foram nomeados assim em homenagem ao matemático George Boole, que definiu um sistema algébrico para determinar o valor de verdade de expressões lógicas. Toda linguagem de programação apresenta alguma representação para valores booleanos, porque muitas vezes, é necessário declarar se algo é ou não é. Em Python, esses valores são as palavras True para verdadeiro e False para falso.

Os valores abaixo são considerados falsos:

  • False (falso);
  • None (nulo);
  • 0 (zero);
  • ”” (string vazia);
  • [] (lista vazia);
  • () (tupla vazia);
  • {} (dicionário vazio).

Observação: Python diferencia capitalização, portanto true, TRUE é diferente de True com a inicial maiúscula.

Operadores Lógicos
Operador lógico E

Às vezes, temos situações em que temos várias condições que necessitam ser verdadeiras ao mesmo tempo. Como por exemplo, para definir de alguém passou de ano, para isso é necessário que ele tenha tirado nota acima da média em todas as matérias, ou seja, tenha tirando uma nota maior que (>=) 6 em matemática, uma nota maior que (>=) 6 em português e assim por diante em todas as matérias. Nessas situações usamos o operador lógico and (E), operador lógico que só retorna verdadeiro se todas as expressões forem verdadeiras.

Operador lógico OR

Em outros casos pode ser que tenhamos uma situação que seja necessário apenas uma condição verdadeira para executar. Como por exemplo, um aluno não precisa ir na escola se for sábado ou domingo, certo? Nessas situações usamos o operador lógico or (OU), operador lógico que só retorna falso se todas as condições forem falsas.

Operador lógico NOT

O operador lógico NÃO, diferente dos operadores lógicos a única função é inverter o resultado, tornando o que era verdadeiro em falso e falso em verdadeiro.

Observação: Ao contrário dos operadores E e OU, o operador NÃO vem antes da comparação.

Revisão dos operadores lógicos

No operador and(E), o resultado só será verdadeiro caso todas as comparações sejam verdadeiras. Ou seja, se qualquer condição que compõe a linha o E for falsa, o resultado será falso.

No operador or(OU), o resultado será verdadeiro com apenas uma das comparações sendo verdadeira. Ou seja, um conjunto de várias condições interligadas por OU será verdadeira no caso de pelo menos uma delas ser verdadeira.

No operador not(NOT), o resultado será o inverso de uma determinada condição. Ou seja, ele irá inverter o resultado de forma que o verdadeiro passe a valer falso e falso tenha valor de verdadeiro.

Estruturas condicionais

Até o momento, vimos que o algoritmo executa uma linha após a outra, executando todas as linhas do início ao fim. Porém, existem casos em que uma linha só pode ser executada a partir de algo.

Os códigos de um programa normalmente são executados de forma sequencial, ou seja, o código da linha de baixo será executado logo após o código de cima. No entanto, às vezes é necessário alterar essa ordem de execução, para por exemplo realizar algo apenas em determinada situação, é aqui que os operadores relacionais estudados anteriormente serão muito utilizados.

As estruturas condicionais realizam essa tarefa de controlar o fluxo de execução do programa, utilizando de teste lógico simples e verificando se é verdadeiro ou falso. Quando o código a ser executado só tem uma ramificação, pode-se utilizar o comando if ou se em português. Ele irá verificar se se determinada condição é verdadeira, se for executará o código, um exemplo simples seria: se a pessoa tiver mais de 18 anos, informe que ela deverá votar, aqui o comando se, está sendo usado com uma condição do tipo idade > 18 e imprimindo uma mensagem informando que deve votar.

Até agora não precisamos preocupar com esse espaço antes do início da linha, mas agora, todo algoritmo que se deseja executar no if, precisa estar mais para frente da linha anterior.

Existem casos, que se deseja realizar uma ou outra funcionalidade dependendo do resultado do teste lógico, lembrando que pode ser verdadeiro ou falso. Para isso, pode usar o código else ou senão em português, onde irá executar determinado bloco de código caso seja verdadeiro ou então um outro fluxo de comando, como por exemplo: se o aluno tirou mais de 7 na média final, ele está aprovado caso contrário reprovado. Aqui, o operador if está fazendo uma comparação de nota > (maior que) 7 para verificar e informar se o aluno está aprovado e no caso de ser uma nota menor que 7, irá executar os comandos presentes no else e assim informar que o aluno reprovou.

E se o objetivo for melhorar o exemplo anterior e informar se o aluno está de recuperação ou reprovado, como resolver? Isso não é nenhum desafio, já existem estruturas pensadas para solucionar esse problema, são as chamadas else if ou senão se, seu funcionamento é bastante semelhante ao if e ao else separados, porém a diferença aqui é que ao invés de executar o código do else automaticamente, irá ser feito uma nova verificação. No exemplo anterior, uma opção seria: se aluno possuir média maior que 7 está aprovado, senão se a nota for entre 7 e 3 está de recuperação, caso contrário se for menos que 3 reprovado.

Isso tem suas vantagens, pois permite colocar inúmeras comparações em sequência e quando uma delas for verdadeira as verificações seguintes não são realizadas.
